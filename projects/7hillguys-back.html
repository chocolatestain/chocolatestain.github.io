프로젝트 개요

청년의 미래 가능성에 투자하는 AI 기반 금융 플랫폼

사람의 가능성에 투자하다
Peo + ch = People + 같이/가치

기술 스택

  • 프론트앤드: React, JavaScript
  • 백엔드: Spring Boot, Java
  • 데이터베이스: MySQL
  • 배포: Jenkins, Linux 자체 서버
  • 버전 관리: Git, GitHub

개발 기간

2025.01 ~ 2025.03

주요 기능

  • 사용자 커스텀 카드(혜택, 디자인) 제작
  • 사용자 정보를 통한 AI 사용자 소득 예측
  • 도출해낸 예상 소득에 기반한 고객 미래 투자
  • 사용자 성장 지원

개발 과정 및 역할

애자일 방법론을 사용하여 개발을 진행했으며, 저는 백엔드 개발을 전담하여 API 설계 및 구현, 데이터베이스 연동, 배포 자동화 등을 담당했습니다. 특히 Jenkins를 활용한 CI/CD 파이프라인 구축에 기여했습니다.

도전 과제 및 해결

대용량 트래픽 처리를 위한 데이터베이스 최적화와 보안 취약점 개선이 주요 도전 과제였습니다. N+1 문제 해결을 위해 Fetch Join을 적극 활용하고, JWT 토큰의 보안 강화를 위해 Refresh Token을 도입하여 해결했습니다.

성과 및 배운 점

안정적인 백엔드 시스템을 성공적으로 구축하여 프론트엔드 개발팀과의 협업을 원활하게 진행할 수 있었습니다. Spring Boot의 다양한 기능을 깊이 이해하고, CI/CD 파이프라인 구축 경험을 통해 개발 생산성을 향상시키는 방법을 배웠습니다.
