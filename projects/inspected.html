프로젝트 개요

inspected는 Django 기반의 웹 애플리케이션으로, 팀 프로젝트로 진행된 캡스톤디자인 과제입니다.
이 프로젝트는 사용자 인증, 게시판, 파일 업로드, 데이터베이스 연동 등 웹 서비스의 기본적인 기능을 구현하는 것을 목표로 하였습니다.

주요 기능

  • 회원가입 및 로그인/로그아웃
  • 게시판(글 작성, 수정, 삭제, 목록, 상세보기)
  • 파일 업로드 및 다운로드
  • MySQL 데이터베이스 연동
  • 관리자 페이지를 통한 데이터 관리

기술 스택

  • 백엔드: Python, Django
  • 데이터베이스: MySQL
  • 프론트엔드: HTML, CSS, JavaScript (Django 템플릿)

담당 파트

  • 웹 서비스 전체 구조 설계 및 구현
  • 사용자 인증 및 권한 관리
  • 게시판 및 파일 관리 기능 개발
  • DB 모델링 및 연동
